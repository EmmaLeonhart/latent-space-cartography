1. What IS Vector Symbolic Architecture?

VSA is a family of computing frameworks that represent complex data structures as high-dimensional vectors and manipulate them with a small set of well-defined operations. The core idea: you can do symbolic reasoning (logic, variable binding, composition) using vector arithmetic in high-dimensional spaces.

The Three Universal VSA Operations

+
Bundling
Superposition of vectors. "This AND this." Result is similar to all inputs.
⊗
Binding
Pairs two vectors. "This WITH this." Result is dissimilar to all inputs.
π
Permutation
Reorders elements. "This THEN this." Encodes sequence/order.

Key Algebraic Properties

  • Bundling is commutative, associative, and the result is similar to all inputs (you can tell what went in)
  • Binding is (usually) commutative, associative, and the result is dissimilar to all inputs (the pair doesn't look like either component)
  • Binding has an inverse (unbinding): if C = A ⊗ B, then A ≈ C ⊗ B¹

Formal Requirements for Binding (Plate 2003)

  1. Dissimilarity: δ(A ⊗ B, A) ≈ 0 and δ(A ⊗ B, B) ≈ 0
  2. Invertibility: Given C = A ⊗ B and B, you can recover A approximately
  3. Distributes over bundling: A ⊗ (B + C) ≈ (A ⊗ B) + (A ⊗ C)
Critical point: Addition FAILS requirement #1. The sum A + B is similar to both A and B — that's what makes it bundling, not binding. This is why our operations are bundling/unbundling, not binding/unbinding.

2. The VSA Variants

Each variant implements the three operations differently. All share the same algebraic structure.

FHRR: The Closest Variant to Our Work

In FHRR, binding IS angle addition and unbinding IS angle subtraction. Since mxbai-embed-large produces L2-normalized vectors (‖v‖ = 1), all vectors lie on the hypersphere, and cosine similarity measures angular relationships. Our relational displacement analysis operates in exactly the angular domain where FHRR binding lives.

3. The TransE ↔ VSA Bridge NOVEL — not in existing literature

These two research communities (KG embeddings and VSA/HDC) developed independently. Nobody has published this correspondence.

KG Embedding Scoring Function VSA Equivalent Operation
TransE (Bordes 2013) ‖h + r − t‖ No direct equivalent — bundling-like Addition
RotatE (Sun 2019) ‖h ⊙ r − t‖ (complex) = FHRR binding Angle addition
HolE (Nickel 2016) Circular correlation = HRR binding Circular convolution
DistMult (Yang 2015) h ⊙ r ⊙ t = MAP binding Element-wise multiply
ComplEx (Trouillon 2016) Re(h ⊙ r ⊙ t̅) ≈ FHRR binding Complex multiply
Implication: The KG embedding literature has been independently reinventing VSA variants for a decade. TransE is the odd one out — it uses addition (bundling), which is why it can't model symmetric relations (a known VSA property: bundling is commutative, so A + B = B + A, no directionality).

4. Our Operations Mapped to VSA

Our Operation Math VSA Interpretation
Displacement extraction d = f(tail) − f(head) Unbundling: removing head from superposition to isolate relational residual
Prediction f(head) + d̄p ≈ f(tail) Rebundling: superposing entity with relational signal to predict target
Two-hop composition f(head) + d̄p1 + d̄p2 ≈ f(tail) Sequential bundling: works because addition is associative
Consistency metric mean cos(gi, d̄p) Signal coherence: SNR of the bundled relational component

NOT TransE

  • TransE trains vectors so h + r ≈ t
  • Single model, single KG
  • Relations are learned parameters
  • Composition not supported

What We Actually Do

  • We observe h + d ≈ t in frozen embeddings
  • Three models, same structure emerges
  • Relations discovered by averaging observed displacements
  • Composition works: d1 + d2 chains naturally

Why This Matters

The finding is: text embedding models spontaneously learn to bundle relational information into their vector representations, and this bundled information can be extracted and composed without any relational training.

5. What's Novel (In VSA Terms)

Things nobody has done before

  1. Shown emergent VSA-like operations in frozen text embedding outputs. McCoy et al. (2019) found TPR in RNN internals. "Attention as Binding" (2025) found binding in attention mechanisms. Nobody has shown the output embeddings exhibit extractable relational structure consistent with VSA.
  2. Connected TransE-family KG embeddings to VSA algebraically. The bridge table above does not exist in the published literature.
  3. Used VSA-style analysis to discover a production defect. The [UNK] collision = destruction of algebraic capacity. Collided embeddings lose their identity vectors, making all VSA operations impossible.
  4. Demonstrated bundling-based relational extraction across 3 architectures. 30 universal operations. Substrate-independent — a hallmark of VSA.

Honest Positioning — What to Claim and What Not To

Do NOT claim:

  • ✗ "Frozen embeddings implement VSA binding" (fails Plate's dissimilarity requirement)
  • ✗ "We have a complete VSA" (no permutation, no random codebook)
  • ✗ "This replaces KG embedding training" (our MRR is lower than trained KGE)

DO claim:

  • ✓ "Frozen embeddings encode relational structure as bundled superpositions consistent with VSA principles"
  • ✓ "Relational displacement is unbundling — extracting the relational component from a superposition"
  • ✓ "Composition works because bundling is associative"
  • ✓ "The consistency metric measures signal-to-noise of the bundled relational component"
  • ✓ "This connects two independent literatures (KGE and VSA) that describe the same mathematical structures"
  • ✓ "Domain-specific probing via this VSA lens reveals production defects invisible to standard benchmarks"
